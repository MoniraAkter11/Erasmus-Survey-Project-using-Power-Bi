# Erasmus-Survey-Project-using-Power-Bi

This project is about the survey we have conducted in Hungary during time of our training Program of Erasmus+.
We Worked as a team their to conduct survey on Erasmus+ youth policy. Made a dashboard project on this survey research.

<br>
Created an interactive Power BI dashboard based on Erasmus+ Survey responses. Visualized key insights, including age distribution, gender analysis, youth engagement, and perceived benefits of youth services. Included charts like bar graphs, pie charts, and a tree map to provide actionable insights. Highlighted correlations between education level, age groups, and youth participation in programs.
